Plotting

Here we can start putting all the plotting tools, and actual example plots that look good and not those tiny weird looking jupyter notebook plots.

We’ll still use the exact same routines and codes.

Single Receiver Function

Given the data downloaded using the example notebook for data collection, we can simply read and plot them using

 1from pyglimer.plot.plot_utils import set_mpl_params
 2from pyglimer.plot.plot_utils import plot_single_rf
 3from pyglimer.rf.create import read_rf
 4set_mpl_params()
 5
 6# Read all RFs from Station IU/HRV
 7rfst = read_rf("../database/waveforms/RF/P/IU/HRV/*.sac")
 8
 9# Some random RF from the 800 avalable one at IU.HRV
10N = 753
11
12# Plot RF and save its output.
13plot_single_rf(rfst[N])

which results in the following image:

We can time limit the figure as well

plot_single_rf(rfst[N], tlim=[0, 20])

which cuts out the RF between 0 and 20 seconds

If you feel artsy and only want the trace

plot_single_rf(rfst[N], tlim=[0, 20], clean=True)

which removes all labels, axes etc.

Receiver Function Section

We can plot all receiver functions in an RFStream into a section depending on epicentral distance.

# Plot section
plot_section(rfst, scalingfactor=1)

This plots all available RFs in the Stream into a section

Also this plot can be limited using the right arguments

1# Plot section with limits
2timelimits = (0, 20)  # seconds
3epilimits = (32, 36)  # epicentral distance
4plot_section(rfst, scalingfactor=0.25, linewidth=0.75,
5             timelimits=timelimits, epilimits=epilimits)

which provides a more detailed view of the receiver functions

CCP Stack Plotting

The stacks computed and saved by the routines have a few built-in plotting tools for simple visualization

Volume Sections

Here we compute a CCP volume first and subsequently slice through it by simply taking slices from a 3D matrix.

 1from pyglimer.ccp.ccp import read_ccp
 2from pyglimer.plot.plot_utils import set_mpl_params
 3import matplotlib.pyplot as plt
 4set_mpl_params()
 5
 6# Read the CCP Stack
 7ccpstack = read_ccp(filename='../ccp_IU_HRV.pkl', fmt=None)
 8
 9# Create spacing
10lats = np.arange(41, 43.5, 0.05)
11lons = np.arange(-72.7, -69.5, 0.05)
12z = np.linspace(-10, 200, 211)
13
14# Plot volume slices: vplot is a VolumePlot object
15vplot = ccpstack.plot_volume_sections(
16    lons, lats, zmax=211, lonsl=-71.45, latsl=42.5, zsl=23)

Cross Section

We can build cross sections without actually compute a full volume. This is done (like the computation of the fiull volume) by using a KDTree that provides weighted nearest neighbour interpolation on a sphere.

 1import numpy as np
 2from pyglimer.ccp.ccp import read_ccp
 3from pyglimer.plot.plot_utils import set_mpl_params
 4import matplotlib.pyplot as plt
 5set_mpl_params()
 6
 7# Read the CCP Stack
 8ccpstack = read_ccp(filename='../ccp_IU_HRV.pkl', fmt=None)
 9
10# Create points waypoints for the cross section
11lat0 = np.array([42.5, 42.5])
12lon0 = np.array([-72.7, -69.5])
13lat1 = np.array([41.5, 43.5])
14lon1 = np.array([-71.45, -71.45])
15
16# Set RF boundaries
17mapextent = [-73.5, -69, 41, 44]
18depthextent = [0, 200]
19vmin = -0.1
20vmax = 0.1
21
22# Plot cross sections
23ax1, geoax = ccpstack.plot_cross_section(
24    lat0, lon0, z0=23, vmin=vmin, vmax=vmax,
25    mapplot=True, minillum=1, label="A",
26    depthextent=depthextent,
27    mapextent=mapextent)
28ax2, _ = ccpstack.plot_cross_section(
29    lat1, lon1, vmin=vmin, vmax=vmax,
30    geoax=geoax, mapplot=True,
31    minillum=1, label="B",
32    depthextent=depthextent
33)
34
35plt.show()
