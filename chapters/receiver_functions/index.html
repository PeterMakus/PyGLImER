RF HandlingÂ¶

Supposing that you have created receiver functions and have them available in the directory output/RF, we can now start imaging!

Handling of receiver functions is generally done by using the RFTrace and RFStream classes. These classes are built upon a modified version of the rf project by Tom Eulenfeld.

Note

Just as in the obspy base classes, an RFStream can hold several receiver functions, while an RFTrace object can only hold one receiver function. As all rfs are saved in .sac format, saving RFStream will lead to the creation of several files.
